* common lisp functions
** cl-remove-if-not
   remains items that satisfy the predicate
   e.g.
   #+begin_src elisp
   (require 'cl-lib)
   (cl-remove-if-not (lambda (x) (> x 3)) '(1 3 2 4 6 0 2 5))
   #+end_src

* setq-default
  https://stackoverflow.com/a/18173666
  allow a variable to have a global value and local values for buffers
  'setq-default' assigns a variable a global value and 'setq' assigns a variable a local value for the current buffer
