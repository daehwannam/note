
* Wi-fi (wifi)
https://blog.rottenwifi.com/ubuntu-connect-to-wifi-terminal/
** wifi
- enable wifi
  #+begin_src sh
  wifi on
  #+end_src
- disable wifi
  #+begin_src sh
  wifi off
  #+end_src
** nmcli
- check wifi status
  #+begin_src sh
  nmcli radio wifi
  #+end_src
- enable wifi
  #+begin_src sh
  nmcli radio wifi on
  #+end_src
  this may require a few seconds
- disable wifi
  #+begin_src sh
  nmcli radio wifi on
  #+end_src
- list SSIDs
  #+begin_src sh
  nmcli dev wifi list
  #+end_src
** nmtui
network manager with text-based UI
- run
  #+begin_src sh
  nmtui
  #+end_src
- key bindings
  - go forward or enable: press <enter>
  - go to the back page: press <escape>
  - moving keys
    - ctrl-n, ctrl-p, ctrl-f, ctrl-b
    - TAB, shift-TAB
- wifi connection
  "Activate a connection" -> select a SSID

  If you cannot see available SSIDs, you should enable wifi beforehand:
  #+begin_src sh
  wifi on
  #+end_src

  If a wifi is connected, '*' is attached to the SSID of connected wifi.
